<h3 align="center">🎯 一款功能强大、界面精美的元素周期表工具，为学生、教师和化学爱好者提供全方位的学习和研究支持</h3>

## 📖 项目介绍

**元素周期表 PM** 是一款专为化学学习和研究设计的现代化 Web 应用工具。本项目采用响应式设计，完美支持桌面电脑、平板电脑和手机等各种设备。通过直观美观的界面和丰富实用的功能，帮助用户深入理解元素周期表的结构、元素性质的周期性变化规律，以及掌握化学相关知识。

### ✨ 核心特性

- 🎨 **精美界面设计** - 采用现代极简主义设计风格，支持深色/浅色主题切换，提供舒适的视觉体验
- 📱 **全平台响应式** - 自适应各种屏幕尺寸，在手机、平板、电脑上都有完美表现
- 🔍 **智能搜索查询** - 快速查找任意元素，支持按原子序数、元素符号、元素名称等多种方式搜索
- 📊 **元素性质展示** - 详细展示每种元素的物理化学性质、电子排布、同位素信息等
- 🎮 **互动学习工具** - 元素猜谜游戏、单位换算器、摩尔计算器等多种学习辅助工具
- 📈 **数据可视化** - 元素性质排序、电子排布模拟等可视化功能，帮助理解抽象概念
- 🌐 **离线可用** - 纯前端实现，无需联网即可使用所有功能
- ♿ **无障碍设计** - 遵循 WCAG 2.1 标准，确保所有用户都能轻松使用

## 🛠️ 技术栈

本项目采用纯前端技术栈，无需后端服务器：

- **前端三剑客**: HTML5 + CSS3 + JavaScript (ES6+)
- **响应式设计**: CSS Grid + Flexbox + Media Queries
- **图标方案**: SVG 矢量图标（无外部依赖）
- **数据存储**: LocalStorage 本地存储
- **部署方案**: GitHub Pages 静态托管

## 🚀 快速开始

### 🌐 在线访问

**GitHub Pages 在线预览版本：**
👉 [https://siweimidu.github.io/PeriodicTablePM/](https://siweimidu.github.io/PeriodicTablePM/)

### 💻 本地运行

1. **克隆仓库**
   ```bash
   git clone https://github.com/siweimidu/PeriodicTablePM.git
   cd PeriodicTablePM
   ```

2. **打开应用**
   - 直接在浏览器中打开 `index.html` 文件
   - 或使用任意本地服务器（推荐）
   
   ```bash
   # 使用 Python 内置服务器
   python -m http.server 8000
   
   # 或使用 Node.js 的 http-server
   npx http-server -p 8000
   ```

3. **访问应用**
   - 浏览器打开 `http://localhost:8000`
