# 元素周期表-鸿蒙网页版

这是一个基于鸿蒙系统的元素周期表网页应用，提供了丰富的化学元素信息和多种实用工具。

## 功能特点

- **元素周期表展示**：直观展示所有化学元素的周期性排列
- **元素详情查看**：点击元素可查看详细的物理和化学性质
- **多种实用工具**：
  - 元素排序功能
  - 摩尔计算器
  - 单位换算器
  - 溶解度表
  - 电子排布模拟器
  - 放射性碳定年法计算器
  - 活化能计算器
  - 元素猜谜游戏

## 技术特点

- 响应式设计，适配多种设备
- 鸿蒙系统优化
- 现代化UI设计
- 丰富的交互效果

## 使用方法

直接在鸿蒙系统浏览器中打开index.html文件即可使用。
